Linux配置swap(虚拟内存)
发表于2020-08-11|Linux开服教程
4G内存以下的服务器建议配置SWAP（也就是虚拟内存）避免内存不足 设置4G的交换分区设置多少由count的值决定 dd if=/dev/zero of=/home/swap bs=1024 count=3764224 创建交换分区文件mkswap /home/swap 启用交换分区文件swapon /home/swap 设置开机启用交换分区文件在**/etc/fstab**中添加home/swap swap swap defaults 0 0 设swappiness不然阿里云，百度云默认是不会使用swapsysctl vm.swappiness=50 永久调整，避免重启之后，swappiness又变成0在**/etc/sysctl.conf**中 vm.swappiness = 0改成vm.swappiness = 50 设置完成之后可输入 free -h 查看内存 结束语至此Linux配置swap完成，在Linux下搭建方舟服务器必须设置swap，否则无法启动服务端！
